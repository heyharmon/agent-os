# ADR 0004: Server-rendered Blade, no SPA framework

The UI is server-rendered Blade. We are not adding React/Vue/a SPA framework.
Interactivity that genuinely needs JS uses small vanilla sprinkles. Proposals
to "rebuild the front end as a SPA" are out of scope.
